# Trivia Quiz
## Descrizione del progetto
Un gioco trivia sviluppato in Flutter.

## Struttura dell'applicazione
L'app è composta da 3 schermate principali:
  * la HOME SCREEN: in cui è possibile selezionare la quantità di domande e la difficoltà di esse;
  * la QUESTION SCREEN: in cui viene mostrata la domanda con le risposte e si deve selezionare la risposta corretta, questa schermata viene visualizzata per la prima volta dopo aver avviato la partita dalla HOME SCREEN e viene aggiornata ad ogni domanda;
  * la RESULT SCREEN: in cui viene visualizzato il punteggio finale, questa schermata appare dopo aver risposto a tutte le domande.

## Per installare l'applicazione su Windows:
### Clonare la repository remota all'interno del computer:
da terminale, digitare: 
```bash
https://github.com/Michele-Bellardi/quiz.git
```
### Avviare il progetto su Windows:
1. aprire il progetto con Android Studio o Visual Studio Code;
2. avviare un emulatore Android;
4. avviare il progetto.

## Per instalare l'applicazione su android:
Dopo aver aperto il progetto su Windows:
1. generare il file .apk da Android Studio;
2. trasferire il file generato all'interno della meoria del cellulare Android;
3. aprire il file .apk dal cellulare e seguire la procedura di installazione indicata;
4. aprire l'applicazione appena installata.
